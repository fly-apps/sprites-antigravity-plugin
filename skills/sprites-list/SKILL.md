# Sprites List

Use the hosted `sprites` MCP server.

1. Call the read-only tool that lists Sprites.
2. Summarize each Sprite with name, status, age if available, service count, public URL exposure if available, and whether it appears plugin- or agent-managed.
3. Highlight likely stale Sprites, but do not destroy anything.
4. If the user needs to choose a target Sprite, recommend the safest matching Sprite and mention the exact name.
