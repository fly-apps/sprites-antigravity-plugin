# Sprites New

Use the hosted `sprites` MCP server.

1. Choose a clear name with the restricted-token prefix, preferably `mcp-`, unless the user provides an exact name.
2. State the purpose before creating the Sprite.
3. Create the Sprite.
4. Wait for it to become usable if the tool reports that it is starting.
5. Run a small verification command such as `pwd` or runtime discovery only if useful.
6. Report the Sprite name and next recommended action.
